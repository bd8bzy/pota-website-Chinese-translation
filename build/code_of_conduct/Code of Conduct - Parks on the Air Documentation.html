    行为准则

    想要参与 POTA 活动，你须要在 https://pota.app/ 网站上注册一个账户。对于违反本行为准则的用户，Parks on the Air 公司将保留临时或永久删除其所有访问权限和数据的权利。管理团队/负责人拥有最终决定权。

    POTA 的注册用户应当：

    1. 遵循 POTA 的规则。
    2. 在 POTA 的任何站点、论坛、社交平台主题上发布相关信息、进行评论。
    3. 其操作遵守 ARRL的操作行为指导。
    4. 其操作遵守DX行为准则。
    5. 其操作遵循所在公园护理员/管理员的指导和决策。

    POTA 的注册用户不应当：

    1. 做出损害空中公园活动或其自身名誉的行为。
    2. 以任何方式威胁、辱骂管理团队、志愿者、其他参与者。
    3. 在 POTA 相关的任何站点、论坛、社交平台上发布下类内容或评论：
      • 粗鲁性的
      • 冒犯性的
      • 恶意争论
      • 侵犯他人版权
      • 发布未经 POTA 许可的产品或服务广告
      • 关于其他公园式野外操作活动的内容
        • 请知悉上述要求并不适用于非公园式野外操作活动，例如 SOTA、IOTA、 BOTA、美国岛屿活动等等。
    4. 以不恰当的方式使用、访问或企图访问 parksontheair.com 或 pota.app 站点。

    违反本准则的用户将会收到一次警告。对于持续的违反行为，管理团队/负责人将会做出适当的处置，可能包括临时性/永久性删除全部访问权限及数据。管理团队/负责人拥有最终决定权。

    如果你对于某个帖子的审核存在争议，请通过 administrator@parksontheair.com 邮箱直接给管理团队发送详情信息。管理团队/负责人拥有最终决定权，并且这是管理团队与用户之间私有事项。
